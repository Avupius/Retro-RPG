# Retro RPG


--- 

## Installation 

1. Python 3.13+ wird benötigt
2. Benötigte Python-Bibliotheken installieren: 
    ```bash
    pip install pygame

    ```


## Spiel starten


1. Schritte im Abschnitt **Installation** befolgen
2. Spiel starten aus Projekt-Root mit:
    - Windows
        ```bash
        python .\Code\main.py
        ```
    - Linux
        ```bash
        python3 code/main.py
        ```
